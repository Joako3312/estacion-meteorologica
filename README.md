## 📂 Estructura del Proyecto
*Organización de archivos y recursos del sistema:*

* 📂 ***analisis_ipynb/*** : Prototipado de datos en Google Colab.
* 📂 ***arduino_ino/*** : Código fuente para el Arduino UNO R3.
* 📂 ***dashboard_pbix/*** : Reportes y métricas en Power BI.

## 🛠️ Tecnologías Utilizadas

### 📊 Análisis y Datos (Club de Ciencias)
* **[Google Colab](https://colab.research.google.com/)** - Procesamiento científico de datos.
* **[MySQL Workbench](https://www.mysql.com/products/workbench/)** - Gestión de base de datos relacional.
* **[Power BI](https://powerbi.microsoft.com/)** - Visualización de métricas climáticas.

### 🐍 Software (Programación III)
* **[Python](https://www.python.org/)** - Lógica central con enfoque en **POO**.
* **[VS Code](https://code.visualstudio.com/)** - Entorno de desarrollo principal.
* **Bibliotecas Estándar**: `datetime`, `math`, `random` y `platform`.

### 🤖 Hardware (Robótica)
* **[Arduino IDE](https://www.arduino.cc/en/software)** - Programación de microcontrolador en **C++**.
* **[Tinkercad](https://www.tinkercad.com/)** - Simulación de circuitos y sensores.
* **Componentes**: Arduino UNO R3, sensores de temperatura, humedad y presión.
